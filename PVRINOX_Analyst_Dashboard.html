What is driving the gap between CLSA ₹1,920 and PL Capital ₹1,274?
Target price bridge — key assumption differences by house
Three factors that explain the TP spread
Factor 1 — Valuation multiple
Range: 9.5x to 13x pre-IndAS EV/EBITDA. CLSA uses 10x on FY27 (higher base year), Nuvama uses DCF which implicitly values at a premium. PL Capital at 9.5x and Investec at 12.5x reflect different views on structural rerating potential post-deleveraging.
Factor 2 — Base year EBITDA
CLSA uses FY27 EBITDA of Rs 24.7bn vs PL Capital's Rs 10.6bn (pre-IndAS) and Kotak's Rs 10.3bn. The gap is entirely the Ind AS 116 treatment — post-IndAS numbers are ~2.5x higher. Houses using post-IndAS base get higher absolute EV, but must use lower multiples to be consistent.
Factor 3 — Net debt assumption
CLSA deducts Rs 60bn net debt (includes Ind AS 116 lease liabilities of ~Rs 57bn) from EV, which significantly clips equity value. Houses using pre-IndAS net debt of ~Rs 3.5–5bn deduct far less, giving higher equity value per share for same EV.

Why does Nuvama (EPS ₹78–97) get only ₹1,605, while CLSA (EPS ₹51–75) gets ₹1,920?
Short answer
Nuvama uses a DCF — high EPS × reasonable discount rate = moderate TP. CLSA uses EV/EBITDA on post-Ind AS numbers — large EBITDA × 10x = very large EV. The lease debt deduction does not fully cancel the EV gain.
The Nuvama EPS paradox
Nuvama's FY27E EPS of ₹78 implies a P/E of ~20x at TP ₹1,605. That is modest. The high EPS comes from aggressive revenue (Rs 80bn), faster debt repayment lowering interest, and lower depreciation — not from a higher EBITDA multiple.
Why CLSA TP is higher
CLSA's post-Ind AS EBITDA ₹24.7bn × 10x = EV of ₹247bn. Deduct ₹60bn net debt (incl. leases) → ₹187bn equity → ₹1,920/share. Nuvama DCF arrives lower because it discounts future free cash flows, not a spot multiple on inflated EBITDA.
Key insight: EPS and TP are not directly comparable across houses when valuation methods differ. An EV/EBITDA house can show lower EPS yet arrive at a higher TP if it uses a post-Ind AS EBITDA base and a spot multiple rather than discounting cash flows.

Screen addition model — FOCO vs FOFO vs owned
FOCO — Franchise Owned, Company Operated
Franchisee builds and owns the property. PVR INOX operates it and takes a revenue share. Zero capex for PVR INOX. Preferred model for new screen additions post-merger. Accounts for bulk of FY27 pipeline.
FOFO — Franchise Owned, Franchise Operated
Franchisee owns and operates under PVR INOX brand. PVR INOX earns licensing/brand fees. Minimal capex. Used in Tier 2/3 cities where local operators have better ground knowledge.
Owned / Leased (legacy model)
PVR INOX builds, leases, and operates. High capex (Rs 4–6 cr per screen), generates long-term lease liabilities under Ind AS 116. Being phased out as primary growth vehicle. Existing base ~1,540 screens.
Capex per new screen: Owned/Leased Rs 4–6 cr | FOCO Rs 0 (franchisee bears) | FOFO Rs 0. PVR INOX guided for Rs 2.5–3bn capex in FY27, almost entirely maintenance + fit-outs for existing screens.

Content pipeline — FY27 analyst expectations
Hollywood slate (bullish)
Mission Impossible 8, Avatar 3, Avengers 5, Superman reboot, Jurassic World 4. After a weak 2 years post-COVID strikes, Hollywood studios normalising output in FY27. CLSA and Nuvama explicitly factor this into upside.
Hindi slate (solid)

Screen share — organised multiplex market
Structural tailwinds analysts cite
Screen density massively underpenetrated
India has ~8 screens per million population vs China 85, USA 125. Long runway to add screens in Tier 2/3 cities without cannibalising existing screens.
Rising income + aspirational spending
Out-of-home entertainment spending growing at 12–15% pa. Multiplex cinema has premiumised into an experience category — Rs 293 ATP still affordable vs Rs 500+ restaurants.
Single screen to multiplex conversion
~200–300 single screens close every year. Most audiences upgrade to multiplex — Carnival/Miraj weakness benefits PVR INOX. No new entrant can replicate PVR INOX's 1,791-screen network economics.
F&B as a separate growth engine
SPH growing from Rs 146 towards Rs 200. Food courts, in-seat delivery, alcohol in select states. F&B margins ~70% gross — higher than ticketing. Nuvama and ICICI Sec model this as the key margin expansion lever.

export default {
  async fetch(request, env) {
    const cors = {
      'Access-Control-Allow-Origin': '*',
      'Access-Control-Allow-Methods': 'POST, OPTIONS',
      'Access-Control-Allow-Headers': 'Content-Type',
    };
    if (request.method === 'OPTIONS')
      return new Response(null, { headers: cors });
    if (request.method !== 'POST')
      return new Response('Not allowed', { status: 405 });
    try {
      const body = await request.json();
      const apiKey = body._apiKey;
      delete body._apiKey;
      if (!apiKey || !apiKey.startsWith('sk-ant-'))
        return new Response(JSON.stringify({error:{message:'Bad key'}}),
          {status:400, headers:{...cors,'Content-Type':'application/json'}});
      const resp = await fetch('https://api.anthropic.com/v1/messages', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'x-api-key': apiKey,
          'anthropic-version': '2023-06-01',
        },
        body: JSON.stringify(body),
      });
      const data = await resp.json();
      return new Response(JSON.stringify(data), {
        status: resp.status,
        headers: {...cors, 'Content-Type': 'application/json'}
      });
    } catch(err) {
      return new Response(JSON.stringify({error:{message:err.message}}),
        {status:500, headers:{...cors,'Content-Type':'application/json'}});
    }
  }
};
